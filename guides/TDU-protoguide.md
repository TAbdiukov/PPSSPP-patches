# How to install

## Prerequisites
* Get a ISO editor, such as UltraISO or UMDGen
* Make sure emulator is off

## Installation
1. Open your TDU ISO file with a ISO editor
2. Go to `{ISO root}/PSP_GAME/USRDIR/SKINNY`
3. Replace .INI files inside with the patched files
4. Save
5. Start the game
